# 🛒 SwapMart

> > **A Basic JavaFX-based Marketplace Desktop Application for Buying, Selling, and Exchanging Second-Hand Products**

---

## 📖 Project Overview

**SwapMart** is a desktop marketplace application designed to simplify the process of **buying, selling, and exchanging pre-owned products**. Built with **JavaFX** and **MySQL**, the application provides a modern and intuitive interface where users can register, log in, manage product listings, browse available products, search items, and connect with buyers and sellers.

The project is developed using **Java**, **JavaFX**, **FXML**, **Scene Builder**, **JDBC**, and **Maven**, following a modular architecture that ensures scalability, maintainability, and a seamless user experience.

---

## ✨ Key Features

- 🔐 **Secure User Login**
  - User authentication with username and password.

- 👤 **User Management**
  - Register new users with profile photo.
  - Update, search, and delete user information.
  - Automatic User ID generation.
  - Support for Buyer, Seller, and Buyer & Seller roles.

- 🛒 **Product Listing**
  - Add products for sale or exchange.
  - Upload front and back product images.
  - Categorize products.
  - Specify product condition, description, expected exchange item, and price.

- 🔍 **Advanced Product Search**
  - Search products by category and product name.
  - View detailed product information.
  - Display product images and availability status.

- ❤️ **Buyer Interest Module**
  - Buyers can submit interest requests.
  - Store contact details and custom messages.
  - Update, search, and delete interest records.

- 🤝 **Secure Deal Management**
  - Create deals between buyers and sellers.
  - Support for **Cash**, **UPI**, and **Product Exchange** transactions.
  - Maintain deal notes and product status.

- 📋 User Records
  - View all registered users.
  - Filter records by user role.
  - Export user data to Excel.

- 📦 Product Records
  - View all listed products.
  - Filter products by category.
  - Display product images and availability.
  - Export product data.

- 📑 Deal Records
  - View complete transaction history.
  - Filter deals by Buyer ID.
  - Export deal records to Excel.

- 📨 Interest Reports
  - Manage buyer interest requests.
  - Export reports to Excel.

- 📊 Analytics Dashboard
  - Visualize user role distribution using Pie Charts.
  - Display product category statistics using Bar Charts.

- 💾 MySQL Database Integration
  - Store users, products, interests, and deals securely using MySQL and JDBC.

- 🎨 Modern JavaFX Interface
  - Responsive desktop GUI designed using JavaFX and Scene Builder.
